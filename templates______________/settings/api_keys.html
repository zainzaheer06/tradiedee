API Usage

Endpoint

POST {{ request.host_url }}api/v1/calls/outbound

cURL Example

curl -X POST {{ request.host_url }}api/v1/calls/outbound \
  -H "Content-Type: application/json" \
  -H "X-API-Key: YOUR_API_KEY" \
  -d '{
    "agent_id": 12,
    "phone_number": "+966501234567",
    "context": {
      "customer_name": "Ahmed",
      "order_id": "ORD-456"
    }
  }'

Python Example

import requests

response = requests.post(
    "{{ request.host_url }}api/v1/calls/outbound",
    headers={"X-API-Key": "YOUR_API_KEY"},
    json={
        "agent_id": 12,
        "phone_number": "+966501234567",
        "context": {
            "customer_name": "Ahmed",
            "order_id": "ORD-456"
        }
    }
)

print(response.json())
# {"success": true, "call_id": 789, "room_name": "api-12-...", "status": "initiated"}

# Demo Data

This directory contains sample/demo data for testing the application.

## Files

### 📄 demo_contacts.csv
Sample contact list for campaign testing
- Format: CSV with contact information
- Use with campaign worker for testing bulk calls

### 📄 demo_knowledge_base.txt
Sample knowledge base content for RAG testing
- Test data for knowledge base functionality
- Upload via the knowledge base interface

## Usage

**Campaign Testing:**
```bash
# Upload demo_contacts.csv through the campaign management UI
# Create a test campaign and import contacts
```

**Knowledge Base Testing:**
```bash
# Upload demo_knowledge_base.txt to an agent's knowledge base
# Test RAG responses with the demo content
```
